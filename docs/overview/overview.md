# Introduction

My Design System은 일관된 제품 경험을 만들기 위한 디자인 원칙, UI 컴포넌트, 패턴, 리소스를 제공하는 디자인 시스템입니다.

이 사이트는 디자이너, 개발자, 기획자가 동일한 기준으로 제품을 설계하고 구현할 수 있도록 돕습니다.

## What’s included

- Foundation: Color, Typography, Spacing 등 기본 시각 체계
- Components: Button, Text input, Select, Data table 등 UI 컴포넌트
- Patterns: Empty State, Loading, Error Handling 등 반복 사용되는 UX 패턴
- Resources: Figma Library, Design Tokens, Release Notes
