## Static

## Scrim

배경 콘텐츠를 눌러주고 상위 레이어를 부각하기 위한 반투명 오버레이 컬러입니다. 모달이나 시트가 열렸을 때, 뒤쪽 화면과의 위계를 분리하는 용도로 사용합니다.

  { color: "rgba(0,0,0,0.5)", name: "--sol-scrim", token: "--color-alpha-black-50" },
]} />
